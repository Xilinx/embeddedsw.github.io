Data Structures
Here are the data structures with brief descriptions:
XDc_AVModesThis typedef stores the data associated with the Audio Video input modes
XDc_BlenderThis typedef stores all the attributes associated to the Blender block of the DisplayPort Subsystem
XDc_ChromaKeyThis structure stores the Chroma keys max, min value for each component
XDc_ConfigThis structure stores the XDC configuration information
XDc_CursorThis typedef describes the cursor blend coords
XDc_PartialBlendThis typedef describes the parital blend coords
XDc_VideoTimingThis structure stores the XDC Timing configuration
XDcDmaThis typedef defines the DCDMA configration
XDcDma_AudioBufferThis typedef is the information needed to transfer Audio info
XDcDma_AudioChannelThis typedef defines the Audio Channel attributes
XDcDma_ChannelThis typedef defines the Video/Graphics Channel attributes
XDcDma_Channel_DscrThis typedef defines the Video/Graphics Channel attributes
XDcDma_FrameBufferThis typedef is the information needed to transfer video info
XDcDma_SDPBufferThis typedef is the information needed to transfer SDP info
XDcDma_SDPChannelThis typedef defines the SDP Channel attributes

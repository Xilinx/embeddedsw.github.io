XAie_LocType  Loc,
XAie_ModuleType  Module 
)

This function is used to check for module and tiletype combination.

Parameters
DevInst,:Device Instance
Loc,:Location of the AIE tile.
Module,:XAIE_MEM_MOD - memory module XAIE_CORE_MOD - core module XAIE_PL_MOD - pl module
Returns
XAIE_OK for correct combination of Module and tile type XAIE_INVALID_ARGS for incorrect combination of module and tile type
Note
Internal API only.

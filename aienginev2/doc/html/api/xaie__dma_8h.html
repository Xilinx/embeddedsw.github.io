Macro to declare DMA queue configuration variable.

Parameters
_Config,:XAie DMA Queue configuration variable name
_StartBD,:Start BD of the Queue
_RepeatCount,:Repeat count
_EnTokenIssue,:XAIE_ENABLE to issue token when completes, otherwise disable.
_OutOfOrder,:XAIE_ENABLE to indicate it is out of order mode, otherwise it is not out of order. If out of order mode is enabled, it will ignore the _StartBd setting.
Returns
None.
Note
The macro declares as an XAie_DmaQueueDesc stack variable.
